门牌尺寸修复测试

修复内容

问题原因：
  • aspect-ratio: 3.6 / 1 限制了门牌的宽高比，导致显示很小
  • background-size: contain 让图片保持比例但可能显示很小
  • 响应式断点的尺寸设置不够大
修复前：
width: clamp(320px, 70vw, 720px);
aspect-ratio: 3.6 / 1;
background-size: contain;
修复后：
width: clamp(400px, 85vw, 900px);
height: clamp(120px, 25vh, 250px);
background-size: cover;

响应式尺寸设置

不同屏幕尺寸的门牌尺寸：
  • 默认： 宽度 400px-900px，高度 120px-250px
  • 超小屏 (≤390px)： 宽度 350px-500px，高度 100px-150px
  • 平板/桌面 (≥768px)： 宽度 500px-1000px，高度 150px-300px

技术细节

关键修改：
  • 移除了 aspect-ratio 限制，允许门牌有更大的高度
  • 将 background-size 从 contain 改为 cover，让图片填满容器
  • 增加了 height 属性，确保门牌有足够的垂直空间
  • 优化了响应式断点的尺寸设置
